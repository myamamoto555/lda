# Latent Dirichlet Allocation (LDA)
This is an implementation of LDA. 

# Installation
## Requirements
- Python2.7
- numpy

## Quick start
```
$ git clone https://github.com/myamamoto555/lda
$ cd lda
$ python lda.py
```
